# 急速监控工具
## 目的
迅速、token消耗友好地查看前后端开发日志
## 实现方法
对于项目中的前后端或者其他业务开发：
要在此文件夹下logs文件夹输入日志信息
然后调用check_log.cmd/check_log.sh --查询参数 进行查看
默认查询：今日的日志文件
查询参数如下：
- 模块
- 日期
- 首行数
- 末行数

## 日志管理
manage_log.cmd/.sh用于管理日志
--clear -before d7 清理七天以外的日志
-using 清理正在使用的日志
-mlm 30kb 30KB以上的日志
总而言之可以通过各种参数来控制清理怎样的日志
--zip (name) 将指定条件的日志打包并添加到（名称）压缩包下
--filter 只展示包含指定字段的
